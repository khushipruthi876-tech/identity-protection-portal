About This Project

The Digital Identity Protection Portal is a cloud-based web application that helps users understand and reduce their exposure to online identity theft. It checks whether a user's email has appeared in known data breaches, analyzes their public digital footprint, and generates a personalized risk report with actionable recommendations.

This project was built as part of a Cloud Technology & Information Security curriculum to demonstrate practical use of cloud-native, serverless architecture for a real-world security use case.

Tech Stack

Frontend

HTML5, CSS3, Vanilla JavaScript

Hosting

AWS S3 Static Website Hosting + CloudFront

Authentication

AWS Cognito

Backend Logic

AWS Lambda (Python) + API Gateway

Database

Amazon DynamoDB

Alerts

Amazon SNS / SES

Project Goals

  • Build a fully serverless architecture on AWS
  • Apply secure authentication and secrets management practices
  • Integrate real-world breach detection APIs
  • Deliver actionable security insights to everyday users
